# Data

This folder contains the datasets used in the Comparative Statistical Analysis project.

## Files

### `dataset1.csv`

Numerical dataset used to explore descriptive statistics, correlation, regression, scatterplots, and boxplots.

### `dataset2.csv`

Numerical dataset used to demonstrate how visual patterns can exist even when Pearson correlation is close to zero.

### `dataset3.csv`

Numerical dataset used to explore a curved relationship where a straight-line regression model only partially explains the pattern.

## Purpose

These datasets are used to compare different types of relationships between numerical variables. The project shows why visualisation is important when interpreting statistical results.

## Note

The datasets are small practice datasets used for portfolio learning and statistical analysis.
