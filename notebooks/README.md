# Notebooks

This folder contains the main analysis notebook for the Comparative Statistical Analysis project.

## Notebook

### `statistical_analysis.ipynb`

This notebook compares three numerical datasets using:

- data loading and inspection
- missing value handling
- descriptive statistics
- quartiles and interquartile range
- Pearson correlation
- simple linear regression
- R-squared
- scatterplots
- boxplots
- cross-dataset comparison

## Purpose

The notebook demonstrates how Python can be used to analyse numerical datasets and why visualisation is important when interpreting statistical relationships.
